# MèchePro

## Comment Installer Django
### Windows
Créér un environnement virtuel (venv)
```python
python -m venv .venv
```
Activer l'environnement virtuel
```python
.venv\Scripts\activate.bat
```
Installer Django
```python
pip install Django
```
## Lancer le serveur
```python
cd mechepro
python manage.py runserver
```
